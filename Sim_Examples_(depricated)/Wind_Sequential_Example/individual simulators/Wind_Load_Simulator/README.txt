- Name of Simulator
Wind_Load_Simulator

- Language or Software
MATLAB R2017b

- Purpose
To obtain the wind force applied on structure at each floor given the wind pressure and the geometry of the building.

- Instructions
Run the script "WindLLoadModel.m" in MATLAB programming environment.
Refer to the script for the definitions of Input and Output Variables.

The default setting is a 20-story building with height = 3.96 m, width = 30.49m, length = 42.68m.
