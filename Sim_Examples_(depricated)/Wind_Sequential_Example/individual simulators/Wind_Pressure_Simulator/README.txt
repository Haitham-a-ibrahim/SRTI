- Name of Simulator
Wind_Pressure_Simulator

- Language or Software
MATLAB R2017b

- Purpose
To obtain the wind pressure applied on structure and cladding at each floor, given the wind speeds and enclosure classification of the building.

- Instructions
Run the script "WindPresModel.m" in MATLAB programming environment.
Refer to the script for the definitions of Input and Output Variables.
The default setting is a 20-story building with height = 3.96 m, width = 30.49m, length = 42.68m.
