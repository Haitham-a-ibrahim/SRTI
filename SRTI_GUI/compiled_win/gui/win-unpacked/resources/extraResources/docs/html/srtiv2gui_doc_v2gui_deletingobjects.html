SRTI v2 GUI - Deleting Obejcts:

This section outlines what you can delete in a SRTI v2 GUI project, and how to do it.


What You Can Delete:

You can delete several types of objects that are visually represented in a project open in the SRTI v2 GUI. These include:

  • Simulators (in Simulator Object List Panel)
  • Messages (in Message Object List Panel)
  • Simulator Objects (in Simulator Canvas)
  • Message Objects (in Message Canvas)
  • Publish Connections (in Simulator Canvas)
  • Subscribe Connections (in Simulator Canvas)

If you delete something accidently, you can use the "Undo" function (see Menu Bar -> Edit -> Undo) to bring it back.

Keep in mind that if you delete an object that is referenced by other parts of the project, those too will be deleted. For example, if you delete a Simulator from the Simulator Object List Panel, any instances of that simulator in the Simulator Canvas will disappear. If you delete a Message Object in the Message Canvas, any Publish or Subscribe Connections to that message will disappear.


How To Delete:

To delete an object, change the action mode to "Delete" mode, and click on the object you wish to delete.
