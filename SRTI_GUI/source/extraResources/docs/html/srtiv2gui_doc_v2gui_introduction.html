SRTI v2 GUI - Introduction:

This section introduces the SRTI v2 GUI.


Brief Overview:

The SRTI v2 GUI (Graphic User Interface) is an optional but recommended component of SRTI v2.00.00. Built in JavaScript and CSS as a desktop application using Electron JS, it's a much more useful and modern-looking GUI than the Java-based one used directly with the RTI Server.

There are 3 main purposes to the SRTI v2 GUI:

  1. To assist in creating and exporting the configuration files the RTI Wrapper(s) rely on, in a visual, more intuitive manner.
  2. To act as a central location to run / launch a simulation system, using a panel to "Play / Pause / Stop" the system with a single click, rather than maintaining many separate simulators by hand.
  3. Saving work made in the v2 GUI as projects, to more easily share simulation systems with other users.

Keep in mind that using the SRTI v2 GUI does not allow a user to avoid understanding the underlying details to how SRTI v2.00.00 functions. It is highly recommended that a user read through the documentation for SRTI v2.00.00 before proceeding to try out the SRTI v2 GUI.
