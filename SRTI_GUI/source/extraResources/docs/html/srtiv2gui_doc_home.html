This is the HTML interactive documentation for SRTI v2 GUI, an optional interface to the open-source, data-transmission and simulation management tool known as SRTI v2.00.00 (part of the "Simple Run-Time Infrastructure" family).

The SRTI software was developed as part of the "Interdepencies in COmmunity Resiliance" (ICoR) research project. Source code and downloads for the SRTI can be found at https://github.com/ICoR-code/SRTI. More information about the ICoR project, its initiatives and team members can be found at https://icor.engin.umich.edu/

(If not already doing so, please open weblinks in a web-browser, and not within the v2 GUI browser.)

(This HTML documentaton is stored locally on your machine alongside the downloaded SRTI v2 GUI. If accessed through the unpackaged version of the v2 GUI executable, you can find the HTML documentation under ".\resources\extraResources\docs\html\" on your computer's file system, from the root location of the v2 GUI executable. Each HTML file can be opened in a standard web browser for your convenience.)
